Глобальные аттрибуты

Нужны для взаимодействия HTML с CSS и JavaScript.

  • Атрибуты
    • id – уникальный элемент для каждого тега
    • class – назначает класс(ы) для CSS и JS
    • style – позволяет писать CSS прямо в теге
    • title – всплывающая подсказка при наведении
    • hidden – полностью скрывает элемент
    • tabindex – порядок перехода по Tab
    • lang – язык содержимого (например, "ru", "en")
    • contenteditable – делает текст редактируемым
    • data-* – собственные атрибуты для хранения данных
    • draggable – можно перетаскивать элемент (true/false)

Ссылки

Тег <a> обозначает ссылку, которую нужно вписать в атрибут href

  • href
    • URL - абсолютный или относительный путь
    • #id - якорь на странице

input

описание

  • <type>
    • text - текст
    • number - число
    • checkbox - выбор нескольких элементов
    • radio - выбобр одного значения из нескольких (для группировки использовать аттрибут name)
    • file - файл
    • другие опции ввода текста:
    • email - почта
    • passord - пароль
    • -
    • url - глобальная ссылка
    • tel - номер телефона
    • search - для поискового запроса
    • дата и время:
    • date - ввод даты в формате (дд.мм.гггг)
    • datetime-local - дата в формате (дд.мм.гггг, чч.мм)
    • month - формат (мм.гг)
    • week - неделя (неделя, год)
    • time - время
    • другое:
    • color - цвет
    • range - полоска ввода
  • аттрибуты
    • value - значение по умолчанию
    • placeholder - показывает текст в пустом поле ввода
    • disabled - нельзя будет взаимодействовать с полем вводаrequired
    • -
    • -
    • multiple - позволяет прикрепить несколько файлов или несколько элементов в выпадающем списке
    • checked - по умолчанию выделенный checkbox и radio
    • min, max - лимиты для number
    • readonly - нельзя редактировать, но можно выделить текст
    • inputmode - задаёт клавиатуру на телефоне
    • autocomplete - включено ли автозаполнение полей, Полезная ссылка
    • form - применяет поле ввода к форме, по id
    • minlength, maxlength - колличество букв в полях ввода
    • pattern - проверяет соответствует ли ввод паттерну, к примеру "[a-zA-Z]+"
    • spellcheck - проверка правописания
  • inputmode
    • text - обычная
    • none - отключает клавиатуру (если есть кастомная к примеру на сайте)
    • numeric - числа
    • decimal - как numeric, но добавится запятая
    • tel - номер телефона
    • email - почта
    • search - поиск
    • url - ссылка

Медиа

Картинки

  • Аттрибуты
    • src - ссылка на изображение
    • alt - будет писаться если фото не загрузилось (стоит писать только если оно важное)
    • width/height - размеры. при загрузке сайта будет заранее выделяться место
    • loading - lazy (отложенная загрузка) или eager (немедленная)
    • decoding - async (асинхронная декодировка), sync (синхронная) или auto
    • srcset - набор изображений для разных разрешений экрана
    • sizes - условия для выбора из srcset

<img src="\ссылка\" alt="нико" height="300"> <img src="../media/art.jpg" alt="прото" height="300" loading="lazy"> <img src="\ссылка\" alt="фапута" height="300" decoding="async"> <img src="#" alt="Арт">

Видео.
Тег <source> нужен для нескольких ссылок на видео, и если одна не загрузилась, то будет попытка загрузить другую. а после него можно добавить тег <p> и написать в нём что то типо "видео не поддерживается"

  • Аттрибуты
    • controls - добавляет элементы управления к видео
    • autoplay - запускает видео
    • loop - повторяет видео
    • muted - без звука
    • playsinline - воспроизведение встроено в страницу (для мобильных)
    • preload - auto (загрузить сразу), metadata (только метаданные), none (не загружать)
    • poster - изображение-заставка перед воспроизведением

<video src="..\media\video.mp4" height="300" controls muted preload="metadata" poster="\ссылка\"></video> <video autoplay loop muted playsinline> <source src="..\media\shark.gif" type="video/gif"> <source src="..\media\shark.mp4" type="video/mp4"> <p>браузер не поддерживает видео</p> </video>

Аудио
работают теги для видео

  • Аттрибуты
    • controls - элементы управления
    • autoplay - автоматическое воспроизведение
    • loop - зацикливание
    • muted - без звука
    • preload - стратегия загрузки (auto/metadata/none)
    • volume - громкость (от 0.0 до 1.0)

<audio controls preload="auto" volume="0.2"> <source src="audio.mp3" type="audio/mpeg"> <source src="audio.ogg" type="audio/ogg"> </audio>
